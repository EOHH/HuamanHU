What is JAVASCRIPT?

JavaScript es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás. (Está bien, no todo, pero es sorprendente lo que puedes lograr con unas pocas líneas de código JavaScript).

What can be done in javascript?

  • Store useful values inside variables. In the example above, for example, we ask you to enter a new name and then store that name in a variable called name.

JavaScript language elements

The objects ==> Javascript has some predefined objects or intrinsic objects such as: Array, Boolean, Date, Function, Global, Math, Number, Object, RegExp, and String. It is also possible to create new objects, with their own methods and properties, adapted to the specific needs of each application.

the arrays ==> Arrays are nothing more than structures to store lists of values, that is, like a named list where we can annotate things. Each annotation is identified by its order number in the array (the list), its index. The total number of spaces to annotate things in the array is its length. In this way, if we have the days of the week written down in an array, the first of all will be the one with index 0 and the last will have index 6, with 7 being the length of the array.

The functions ==> Las funciones como decíamos en la introducción, no son mas que bloques de instrucciones de programa con nombre y que pueden ejecutarse sin mas que llamarlas desde alguna parte de otra función o desde la página HTML, bien sea directamente o mediante eventos
