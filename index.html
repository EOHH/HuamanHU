What is a programming language?

It is a formal language that, by means of a series of instructions, allows a programmer to write a set of orders, consecutive actions, data and algorithms in order to create programs that control the physical and logical behavior of a machine. Through this language, the programmer and the machine communicate, allowing precise specification of aspects such as:

  • What data should a specific software operate?
  • How should that data be stored or transmitted?
  • The actions to be taken by the software depending on changing circumstances?
